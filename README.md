# OIT_Stack Corporate Website

![OIT_Stack Banner](assets/logo.png)

## Overview
**OIT_Stack (OnYX Integrated Tech Stack)** is a premium, high-performance static website designed for enterprise consulting and strategic political analytics. It utilizes a state-of-the-art Vanilla architecture (HTML5, CSS3, ES6 JavaScript) to guarantee flawless SEO ranking, zero-latency load times, and maximum security against backend vulnerabilities.

## Features
- **Vibrant Light Theme:** Custom navy blue and glowing cyan design system.
- **Micro-Animations:** Highly optimized JS-driven Intersection Observers for buttery smooth card fade-ins lazily triggered on scroll.
- **10-Page Architecture:** Fully routed multi-page scale including 5 dedicated service-detail pages tailored directly for Search Engine web-crawlers.
- **Glassmorphism UI:** Advanced CSS styling for frosted-glass overlays and deep soft-shadowing.

## Services Offered
1. Software Development
2. Political Data Analytics
3. Campaign Management
4. Political Surveys
5. Corporate Training

## Running Locally
Because this project does not rely on a bulky framework, running it locally is instant. Simply open a terminal in the project directory and run:

```bash
python -m http.server 8000
```
Then navigate to `http://localhost:8000` in your web browser.
